Routing and Switching

Switching Protocols

  • STP is a layer 2 protocol used to prevent loops in a network topology.

  • VLAN is a method of creating logical networks within a physical network.

Routing Protocols

    • RIP
    • EIGRP
    • OSPF
    • Hybrid
    • BGP

Routing Types

    • Routes are manually configured
    • No automatic updates
    • Routes are learned and updated automatically
    • Examples: RIP, EIGRP, OSPF, BGP
    • Used when no other route to a destination is known
    • Default route is used as a last resort
