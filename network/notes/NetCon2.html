Networking Concepts

Overview

Networking involves the communication of devices over a network, and it requires the use of various protocols and ports to establish connections and transfer data.

Protocols

Protocol Type Description
ICMP Internet Control Message Protocol is used for error reporting and diagnostics. It is commonly used by network devices to send error messages indicating that a requested service is not available or that a host or network is unreachable.
UDP User Datagram Protocol is a connectionless protocol used for transmitting data over the network. It is often used for time-sensitive applications such as audio and video streaming.
TCP Transmission Control Protocol is a connection-oriented protocol used for establishing reliable connections between devices. It is commonly used for web browsing, email, and file transfers.
IP Internet Protocol is responsible for addressing and routing data packets across the network. It is the foundation of the internet and other network protocols are built on top of it.

Connections

Network protocols can be either connection-oriented or connectionless. Connection-oriented protocols establish a reliable connection between devices before transmitting data, while connectionless protocols transmit data without establishing a connection first. TCP is an example of a connection-oriented protocol, while UDP is an example of a connectionless protocol.
